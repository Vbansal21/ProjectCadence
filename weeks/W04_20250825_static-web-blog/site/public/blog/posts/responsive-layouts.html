Responsive Design: Fluid Type & Grid

Design to your content, not to a device zoo. Start fluid, then add guardrails.

Fluid typography with clamp()

/* Tailwind-like custom property approach */
:root { --step-0: clamp(1rem, 0.9rem + 0.5vw, 1.125rem); }
h1 { font-size: clamp(1.75rem, 1.25rem + 2vw, 2.5rem); }

Use fluid steps for body text and headings to avoid abrupt jumps.

Layout with modern CSS Grid

.wrap {
  display: grid;
  grid-template-columns: 1fr min(70ch, 100%) 1fr;
}
.wrap > * { grid-column: 2; }

Center content with a readable measure (~60–75 characters) and let side columns absorb whitespace.

Breakpoints serve content, not devices

@media (min-width: 48rem) { /* wide-ish text columns */ }
@media (min-width: 64rem) { /* two-column article & aside */ }

Pick breakpoints where the layout actually strains, not at arbitrary device widths.
